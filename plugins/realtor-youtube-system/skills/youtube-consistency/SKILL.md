# Consistency Engine

Most channels don't fail on content — they fail because the agent stops. Keep them going. Apply `${CLAUDE_PLUGIN_ROOT}/shared/house-rules.md`.

**Applies the YouTube Doctrine** (`${CLAUDE_PLUGIN_ROOT}/shared/youtube-doctrine.md`) — consistency runs on
the doctrine's **cadence (§15.3: 2/week ideal · 1/week minimum · 3/week only if quality holds)**, the
**weekly workflow (§22.4)** and **monthly workflow (§22.5)**, plus **S.E.A.R.C.H. "C" — Consistency (§3)**:
one strong video a week beats three weak ones. The Brain's cadence wins if set; otherwise default to §15.3.

## Batch-day mode (on demand)
The realtor films best in batches. Given the top ranked ideas:
- Pick the next N (default 4) and ensure each has a ready script (call Script Studio if needed).
- Produce a **filming order + run-of-show**: same-outfit/same-setup groupings, shortest setup changes,
  intros/CTAs to record together, and a simple shot checklist per video.
- Offer to drop a "Filming Day" block on their **Google Calendar**.

## Check-ins (when they show up or ask "am I on track?")
- Compare what's actually shipped (Videos folders + channel) vs their target cadence (Brain if set, else the
  doctrine §15.3: 2/week ideal · 1/week minimum). Hold the line that **consistency beats volume** — one
  strong video a week is a win, not a shortfall (§15.3, §3 "C").
- If they return after a gap, one gentle, encouraging line: "Good to see you — it's been a couple weeks;
  want to pick an idea and knock one out? ~30 minutes." Never guilt-trip; momentum over pressure.

## Plan my month (light planning, doctrine §22.5)
On "plan my month": pull the latest ranked ideas and spread them across their cadence (doctrine §15.3 default
~2/week → ~8 videos/month; 1/week → ~4), **drawing across the 6 content types** (§8) so the month isn't all
one pillar. Per the **monthly workflow (§22.5)**: slot the **market update in the first week**, then mix
relocation / community / map / lifestyle / home tours. Suggest a batch-filming day, offer to put real dates on
Google Calendar, and offer to script the first one. A simple plan in chat — never a spreadsheet.

## 30/60/90 ramp (for new channels — the doctrine's 90-day launch, §22)
A staged plan so beginners aren't overwhelmed, aligned to the doctrine's first-90-days goal (build the habit,
test across content types, collect data — §15.1, §22.1): first 30 days = find footing (publish across the 6
content types §8, don't perfect-optimize), 60 = rhythm (batch + repurpose, keep the cadence), 90 = optimize
(audit analytics + lead feedback, lean into what's working — §22.5, §23.3). Deliver the relevant stage.

## Modes
On-demand (batch day, month plan, ramp, "am I on track?") + Calendar blocks when they want them.
