# Voice — Guide

Goal: make every script sound like **the agent**, not generic AI.

## ⭐ Voice already lives in the AI Brain
The shared Realtor AI Brain (brand persona) already captured the agent's voice and tone. **Default to
that.** Do NOT re-run a voice interview. This step is an **optional refinement** that adds their
*spoken, on-camera* cadence on top of the Brain voice — only if the agent wants an extra-tight match.

- AI Brain HAS voice (normal case) → use it. Optionally offer Path A refinement. Never re-ask.
- AI Brain has NO voice at all (rare) → run the Path B mini-interview, and tell them it'll also feed
  back into their AI Brain so it's captured centrally.

---

## Path A — Optional refinement from existing videos
1. Ask them to paste **2–5 transcripts** (or links) of their best/most "them" videos. Auto-captions ok.
2. Extract only what the Brain voice doesn't already capture: **spoken pacing, on-camera energy,
   verbal catchphrases, filler patterns.**
3. Write 2–3 sample sentences blending the Brain voice + this on-camera cadence; ask "does this sound
   like you on camera?" Iterate, then save it as the **on-camera voice refinement** in the YouTube
   Layer (Part B) — do NOT overwrite the AI Brain voice.

## Path B — Mini-interview (ONLY if the AI Brain has no voice)
1. "In one sentence, how would a happy client describe how you talk?"
2. "More buttoned-up and professional, or casual and funny?"
3. "Any words or sayings you use all the time?"
4. "A creator whose delivery you admire?"
5. "Anything you never want to sound like?"
Then synthesize a voice and recommend adding it to the AI Brain.

## Quality bar
- Specific enough to imitate — capture exact phrases and sentence shapes, not adjectives.
- Never invent a personality. The agent must approve the sample sentences.
- Always respect the AI Brain's hard avoids + Fair Housing risk language.
