# The YouTube Layer — Template

This is **NOT a second brain.** It is a thin add-on that reads the shared Realtor AI Brain and stores
only YouTube-specific fields. Anything in Part A is *read live from the AI Brain* — never re-asked,
never copied here.

---

## Part A — Inherited from the AI Brain (READ live — never ask, never store here)
Listed only so you know what's already available. Source mapping: see `ai-brain-contract.md`.
- Identity — name, brokerage, market(s)/cities
- Niche & positioning + what makes them different
- Avatars / target viewers (their situation, fears, search questions)
- Offer, USP, CTA, booking link
- Voice & brand tone
- Brand kit (colors/fonts/logo) — used for the design-tool handoff
- Existing lead magnets

## Part B — The YouTube Layer (the ONLY things setup captures)
- **Channel URL:** (or "none yet")
- **Existing performance context:** video count, best performers, worst performers (if any)
- **YouTube goal (12-month):** subscribers / leads target *for YouTube*
- **Posting cadence:** realistic commitment (e.g. 1 long-form/week + 3 shorts)
- **Filming time available:** (e.g. one batch day/month)
- **Competitor channels:** 3–7 channels to track (feeds the Outlier engine)
- **Neighbourhood coverage list:** communities to cover systematically (from the AI Brain) — the coverage read tracks progress against this
- **On-camera voice refinement (optional):** spoken cadence captured from video transcripts, layered
  ON TOP of the AI Brain voice — never replacing it
- **YouTube compliance notes:** channel/brokerage-specific disclaimers for descriptions

## Part C — Generated at setup (NOT stored as spreadsheets)
- **Content pillars** — 4–5, derived from the AI Brain's niche + avatars, confirmed with the agent, and
  stored here in the YouTube Layer. They keep idea-generation strategic — invisibly. (No Content Map file.)
- **First ranked ideas** — generated from a quick city + niche scan and delivered to the agent directly
  (chat / first briefing). NOT saved as an Idea Bank sheet — the system regenerates ideas live each week.
- **CTA library** — 3–5 CTA variations generated from the agent's offer + lead magnets, stored here, so every
  video doesn't repeat the identical close. The writing skills (SEO / Script / Lead / Repurpose) rotate through them.
