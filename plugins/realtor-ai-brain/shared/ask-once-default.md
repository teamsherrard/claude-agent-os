# Ask Once → Default-If-Unsure (the Brain's setup principle)

Every Brain phase skill follows this. It's why a brand-new agent never freezes on a blank question —
and why the output stays detailed and strategic even when they barely type. The goal: **the least
typing for the agent, the highest-quality file for the Brain.**

## The rule
1. **Ask each question once**, conversationally, one at a time. Warm, never a form.
2. **If the agent answers → use it verbatim.** Never overwrite a real answer with a guess.
3. **If they hesitate / "I don't know" / "you pick" / "not sure" → generate a sensible default**
   from their Brain (or a real-estate best-practice standard) and let them confirm in one word.
4. **Global escape hatch — honour these the moment the agent says them, in ANY phase, at ANY point:**
   - **"use defaults" / "set the standard ones" / "you decide"** → fill everything remaining with
     full-quality, Brain-personalised defaults and present it as one block to confirm. One word finishes
     the phase.
   - **"skip" / "do this later" / "not sure I have that"** → write a safe, friendly placeholder and move
     on. Never leave a broken or empty file; never block.

## The quality guardrail (NON-NEGOTIABLE)
**A default is NOT a thin stub.** When the agent accepts a default, generate the COMPLETE, professional,
specific version — exactly as detailed, thoughtful, and strategic as if they'd answered every question
themselves. Pull richly from everything already in the Brain (niche, market, avatars, voice, offer,
proof) so the default is *theirs*, not generic boilerplate. **Reducing the agent's effort must never
reduce the output's quality.** Every deliverable a realtor sees should feel custom and worth keeping.

## Tone for the hard / scary parts
Lead with reassurance and a default, not a blank: *"I'll set safe, standard defaults and you just
confirm — no homework."* Make the agent feel guided, not quizzed. They should be able to finish any
phase by accepting defaults, and still get a file that looks like a pro built it for them.
