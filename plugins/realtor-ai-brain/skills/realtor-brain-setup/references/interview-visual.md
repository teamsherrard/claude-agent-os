# Interview Guide — Visual Brand

Keep this light — 3–5 minutes. Most agents either already have brand colors or have no idea.
Both are fine. The goal is to fill `identity/brand-visual.md` enough that graphic skills look
on-brand. If the agent is lost, don't stall the whole setup — capture what you can and note that
they can run the full **Brand Guide** skill later to design a proper identity.

## How to run
- One question at a time, conversational.
- If they don't know, offer the safe default and move on.
- If they upload a logo/headshot, save it to `~/realtor-brain/assets/` and record the path.

## The questions

**V1: Do you already have brand colors? If so, what are they?**
*(Ask for hex codes if they know them; a description like "navy and gold" is fine — convert to
approximate hex.)*
Default if none: "No problem — we'll start neutral: black, dark gold (#B8860B), white. You can
design a full palette later with the Brand Guide skill."

**V2: Do you have a logo? You can upload it and I'll save it to your Brain.**
*(If yes → save to `assets/logo.png` or `.svg`, record path. If no → note "none yet".)*

**V3: Do you have a professional headshot to use in graphics? Upload it if so.**
*(Save to `assets/headshot.jpg`.)*

**V4: Any fonts you use for your brand? (Or a vibe — modern/classic/bold?)**
Default if none: "We'll default to a clean modern pairing and you can refine later."

**V5: Do you have a tagline or a short line you put on your marketing?**
*(Optional — "none" is a valid answer.)*

**V6 (optional): Three words for how your brand should look and feel?**
*(e.g. "bold, high-contrast, premium" — this guides graphic skills. Skip if they're unsure.)*

## Writing the file
Fill `identity/brand-visual.md` from these answers. Always write *something* for colors and fonts
(use the defaults if needed) so downstream graphic skills never hit an empty file.
