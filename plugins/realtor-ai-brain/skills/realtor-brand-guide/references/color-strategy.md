# Color Strategy for Real Estate Brands

This is the core color decision-making framework Claude uses when building brand color systems. It upgrades color selection from "what looks nice" to "what this brand must communicate before anyone reads a word."

---

## The Foundational Question

Before selecting any color, ask: **"What must this brand say before anyone reads a word?"**

Color is not decoration — it is communication. Up to 90% of a first impression is based on color alone. Color increases brand recognition by up to 80%. For a real estate agent whose brand must communicate trust, expertise, and market authority instantly, the palette is the most powerful signal available.

Never ask "what color does the agent like?" Ask what their brand must *say*.

## The 60-30-10 Color Application Rule

Every brand palette must follow this distribution across all touchpoints:

- **60% — Dominant Color:** The background color of the brand world. Sets overall tone. Almost always a neutral: warm white, cream, sand, light gray, or deep charcoal. Appears on page backgrounds, large sections, white space.
- **30% — Secondary Color:** The main brand personality color. The one that carries the emotional message. Appears on navigation, section backgrounds, key brand elements, prominent text, logo.
- **10% — Accent Color:** The high-contrast, attention-directing color. Appears on CTA buttons, links, icons, pull quotes, and small decorative elements. Never use as a large background.

### 60-30-10 Applied Across Touchpoints

| Touchpoint | 60% Dominant | 30% Secondary | 10% Accent |
|---|---|---|---|
| Website | Page background | Navigation, section backgrounds | CTA buttons, links, hover states |
| Social Media | Post background | Text blocks, overlays | Highlight text, borders, icons |
| Business Card | Card background | Logo, name, contact info | Tagline, divider line, logo mark |
| Email Newsletter | Email background | Header section | CTA button, link text |
| Listing Presentation | Slide background | Section headers | Data callouts, key stats |
| Signage / For Sale Sign | Background | Agent name, brokerage | Phone number, website |

## Tints, Shades, and Tones — Building Color Depth

A single hex code is not a palette. Professional brand color systems use tints, shades, and tones of each color to create depth and versatility.

- **Tint:** Color mixed with white → lighter, softer versions (e.g., Navy → Powder Blue). Use for backgrounds, hover states, subtle UI.
- **Shade:** Color mixed with black → darker, deeper versions (e.g., Sage → Forest Green). Use for text, borders, depth.
- **Tone:** Color mixed with gray → muted, sophisticated versions (e.g., Coral → Dusty Rose). Tones dominate 2026 branding — "muted over saturated."

### Color Scale System (5 levels per brand color)

| Scale Level | Description | Typical Use |
|---|---|---|
| 100 (Lightest Tint) | Near-white version | Section backgrounds, hover states |
| 300 (Light Tint) | Soft, light version | Card backgrounds, subtle dividers |
| 500 (Base Color) | The pure core brand color | Logos, primary buttons, key headings |
| 700 (Shade) | Darker, richer version | Text on light backgrounds, borders |
| 900 (Deep Shade) | Near-black version | Heavy text, dark mode backgrounds |

**Example — Sage Green Scale:**
- 100: `#EEF3EE` (near-white sage)
- 300: `#C2D4C2` (light sage)
- 500: `#7C9A7E` (base sage — the brand color)
- 700: `#4A6B4C` (deep sage)
- 900: `#1E3320` (near-black forest)

## Color Harmony Systems

Select the harmony type that matches the brand's emotional tone:

| Harmony Type | How It Works | Emotional Effect | Best For |
|---|---|---|---|
| **Monochromatic** | Single hue in varying tints/shades/tones | Clean, minimal, sophisticated, effortless | Luxury brands, minimalist agents, high-end residential |
| **Analogous** | 2–3 colors adjacent on the color wheel | Natural, calming, harmonious, organic | Coastal agents, eco-friendly brands, lifestyle-focused agents |
| **Complementary** | Two colors directly opposite on the wheel | High contrast, bold, dynamic, memorable | Urban specialists, commercial agents, high-energy brands |
| **Split-Complementary** | One base + two colors flanking its complement | Contrast with more balance; versatile | Most real estate brands — visual interest without jarring |
| **Triadic** | Three colors evenly spaced on the wheel | Vibrant, playful, energetic | Rarely used; best for community-focused or lifestyle brands |

## Mood Board Color Extraction — 5-Step Process

When an agent describes inspiration imagery or provides a mood board, follow this structured extraction:

**Step 1 — Read the Emotional Temperature**
Is the mood board warm (creams, golds, terracottas, wood tones) or cool (blues, grays, whites, silvers)? High-contrast or low-contrast (soft, tonal, muted)? Organic/natural or architectural/geometric?

**Step 2 — Identify the Dominant Background Color (the 60%)**
Almost always a neutral: warm white, cream, sand, light gray, or deep charcoal. This sets the overall brand environment.

**Step 3 — Identify the Primary Brand Color (the 30%)**
The most emotionally resonant color in the imagery — the color that carries the "personality." Examples: recurring navy in architectural photography, sage green in garden imagery, deep plum in luxury interior shots.

**Step 4 — Identify the Accent Color (the 10%)**
The smallest but most impactful color — a pop of gold in jewelry, a flash of coral in a sunset, a bold red door in an otherwise neutral exterior. This becomes the CTA and highlight color.

**Step 5 — Name the Colors Evocatively**
Professional brand guides never use "dark blue" or "light beige." Use evocative names that reinforce the brand story:
- Instead of "Dark Navy" → "Midnight Tide" / "Estate Navy" / "Ink Black"
- Instead of "Beige" → "Warm Linen" / "Sand Dune" / "Driftwood"
- Instead of "Gold" → "Champagne Coast" / "Antique Brass" / "Heathered Gold"
- Instead of "Sage Green" → "Hill Country Sage" / "Coastal Sage" / "Pacific Mist"

## Six Brand Archetypes — Full Palette Profiles

These are the six core real estate agent brand personalities. Identify which archetype fits the agent, then use the matching palette as the starting point.

### Archetype 1: The Luxury Authority
**Who:** Top 1% producers, luxury estate specialists, celebrity agents, boutique luxury brokerages.
**Personality:** Exclusive, discreet, sophisticated, authoritative, timeless.
**Mood Board:** White marble, black steel frames, champagne flutes, luxury hotel lobbies, dramatic architectural lighting, fine leather goods, monochromatic editorial fashion.
**Color Strategy:** Monochromatic or subtle complementary. Maximum negative space. Deep neutrals + single metallic accent. No bright colors. No warm or casual tones.
**Harmony Type:** Monochromatic

| Role | Color Name | HEX | RGB | Usage |
|---|---|---|---|---|
| Dominant 60% | Crisp White | `#FFFFFF` | 255, 255, 255 | All backgrounds, white space |
| Secondary 30% | Onyx | `#1A1A1A` | 26, 26, 26 | Logo, primary text, headers |
| Supporting Neutral | Warm Ash | `#4A4A4A` | 74, 74, 74 | Body text, secondary elements |

| Light Neutral | Marble White | `#F5F3F0` | 245, 243, 240 | Card backgrounds, section fills |

**Typography:** High-contrast serif (Playfair Display, Cormorant Garamond) + clean sans-serif (Montserrat Light)

## Color Code Documentation Standards

Every color in a brand style guide must be documented in multiple formats:

| Format | Used For | Example (Estate Navy) |
|---|---|---|
| **HEX** | Web design, CSS, digital graphics | `#1B3A6B` |
| **RGB** | Digital screens, video, social media | 27, 58, 107 |
| **CMYK** | Print production (business cards, signage) | 75, 46, 0, 58 |
| **Pantone (PMS)** | Exact color matching across vendors | PMS 2766 C |

Always provide HEX and RGB. Note that CMYK and Pantone values should be confirmed by a print vendor — never generate CMYK or Pantone values from scratch as they require physical calibration.

## Standard Color Usage Rules Template

Every brand guide color section must include explicit rules. Adapt this template:

- **Primary Background:** Always use [Dominant Color] as the primary background on all digital and print materials.
- **Text on Light Backgrounds:** Always use [Deep Neutral] for body text on light backgrounds. Minimum contrast ratio of 4.5:1 for accessibility (WCAG AA standard).
- **Text on Dark Backgrounds:** Always use [Dominant Color / White] for body text on dark backgrounds.
- **CTA Buttons:** Always use [Accent Color] for primary CTA buttons. Button text should be [Deep Neutral or White] depending on contrast.
- **Logo Color Versions:** Always provide: (a) full color on white, (b) reversed on the secondary color, (c) single-color black, (d) single-color white.
- **Do Not:** Do not use the accent color as a background for large sections. Do not use more than three colors in a single design element. Do not alter the opacity of brand colors without explicit guidance in the style guide.

## 2026 Color Trends in Real Estate Branding

Keep these current trends in mind when advising agents:

1. **Muted over saturated:** Moving away from bright primaries toward softer, muted tones. Especially true for agents targeting higher price points.
2. **Warm neutrals as primary backgrounds:** Stark white and cool gray being replaced by cream, sand, taupe, warm linen. Add warmth without competing with property photography.
3. **Intentional single accents:** One carefully chosen accent against a neutral foundation — cleaner visual hierarchy, more recognizable brand.
4. **Earthy and organic tones:** Warm whites, soft creams, sand, terracotta feel authentic and human — a response to oversaturation of digital content.
5. **Deep, moody secondaries:** Midnight navy, forest green, deep plum rather than mid-tone blues and greens — feels more premium and distinctive.

## Claude's Color Decision Workflow

When building a brand color system, follow this order:

1. **Identify the archetype** — map agent's target market, personality words, and niche to one of the six archetypes above
2. **Analyze mood board / inspiration** — if provided, use the 5-step extraction process
3. **Select harmony type** — based on archetype and mood board emotional tone
4. **Build the full palette** — 5 colors with evocative names, HEX, RGB, and 60-30-10 roles assigned
5. **Apply 60-30-10** — map each color to its proportion and document across touchpoints
6. **Write usage rules** — explicit do's and don'ts for the palette
7. **Note 2026 trend alignment** — briefly state how the palette aligns with or intentionally diverges from current trends
