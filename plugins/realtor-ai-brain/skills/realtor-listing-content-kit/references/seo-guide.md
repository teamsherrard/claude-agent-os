# SEO Guide — YouTube Listing Video Titles, Descriptions, and Tags

## Title Strategy for Listing Videos

Listing videos rank on two types of search:
1. **Address search** — buyers searching the specific address
2. **Neighbourhood search** — buyers searching "[neighbourhood] homes for sale [year]"

Titles need to serve both. Always include the neighbourhood name AND the city.
Including the address is optional — it's very specific but captures high-intent buyers.

### Title Formulas

| Formula | Example |
|---------|---------|
| Address + neighbourhood + city | "123 Oak Street — [Neighbourhood], [City] Real Estate Tour" |
| Property type + neighbourhood + year | "[X] Bedroom Home for Sale in [Neighbourhood], [City] — [Year]" |
| Price + neighbourhood | "Under $[X] in [Neighbourhood] [City] — Full Home Tour" |
| Feature + neighbourhood | "[Key Feature] Home in [Neighbourhood] [City] — [Year] Listing" |
| Buyer-type + neighbourhood | "Best [Family/Investor/First-Time Buyer] Home in [Neighbourhood] [City]" |

### Title Rules
- Neighbourhood name in every title
- City name in every title
- Year (2026) in at least 3 of the 5 variations
- Under 60 characters when possible — never over 70
- No clickbait — deliver what the title promises
- Price in title is optional — include if it's a compelling value angle

### Primary Title Selection
After generating 5 variations, select the strongest one as the working primary title.
State it clearly: "PRIMARY TITLE: [title]"
This title anchors the description and tags.

## Description Structure

### Full Description (150-300 words)

```
[Opening sentence — primary keyword + most compelling property detail.
This is what shows in search results — front-load the keyword.]

🏠 Book a Private Showing: [LINK]

[2-3 paragraphs covering:
- Property highlights (bedroom count, key features, price)
- Neighbourhood context (schools, amenities, lifestyle)
- Who this home is perfect for]

In this video:
- [Key feature 1]
- [Key feature 2]
- [Key feature 3]
- [Neighbourhood highlight]

📅 Open House: [Date/Time if applicable]
🏠 Book a Private Showing: [LINK]

[Agent name] is a real estate agent in [City] specializing in [neighbourhood/niche].
[1 sentence of local credibility.]

[8 hashtags]
```

### Short Description (condensed)
```
[1-2 sentence property hook with primary keyword]

🏠 Book a showing: [LINK]

[3-4 bullet points: beds/baths, price, top feature, neighbourhood]

🏠 [LINK]

[6 hashtags]
```

## Tags — 5-Tier Hierarchy for Listing Videos

### Tier 1 — Exact Match (2-3 tags)
The precise search someone would use to find this listing.
- "[neighbourhood] homes for sale [year]"
- "[city] real estate [year]"
- "[address]" (optional — very specific)

### Tier 2 — Broad Match (3-4 tags)
Wider variations of the same topic.
- "[city] homes for sale"
- "real estate [city] [year]"
- "[neighbourhood] real estate"

### Tier 3 — Property Type (3-4 tags)
The specific type of home.
- "[city] [property type] for sale"
- "[bedrooms] bedroom home [city]"
- "[neighbourhood] [property type]"

### Tier 4 — Buyer Audience (2-3 tags)
Who this home is for.
- "[target buyer type] homes [city]"
- "buying a home in [city] [year]"
- "[city] [buyer type] real estate"

### Tier 5 — Long-Tail (2-3 tags)
Specific how-to or search queries.
- "homes for sale in [neighbourhood] [city]"
- "[city] home tour [year]"
- "real estate tour [neighbourhood] [city]"

## Hashtags (8 total)

Mix:
- 2 local: #[City]RealEstate + #[Neighbourhood]
- 2 property: #HomesForSale + #[City]Homes
- 2 audience: #[City]Realtor + #RealEstateTour
- 2 broad: #RealEstate + #[Year]RealEstate
